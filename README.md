# 🌍 Tourism - Explore India's Iconic Destinations

A beautiful, interactive web application showcasing India's most famous tourist attractions. Plan your journey and discover the cultural heritage and architectural marvels of India.

---

## ✨ Features

- **🏠 Home Section** - Welcoming landing page with call-to-action
- **❤️ Favourite Places** - Browse a curated list of iconic destinations
- **🖼️ Detailed Views** - Explore destinations with image carousels and comprehensive information
- **📱 Responsive Design** - Built with Bootstrap for seamless experience on all devices
- **🎨 Beautiful UI** - Modern styling with engaging backgrounds and smooth navigation
- **⚡ Interactive Navigation** - Easy-to-use navigation between sections

---

## 🗺️ Destinations Covered

| Destination | Location | Highlights |
|------------|----------|-----------|
| **🕌 Taj Mahal** | Agra | India's most iconic symbol of love and architecture |
| **🛕 Golden Temple** | Amritsar | A highly revered and beautiful Sikh pilgrimage site |
| **👑 Mysore Palace** | Mysore | Historic royal residence with stunning architecture |
| **🕉️ Varanasi Temple** | Varanasi | Sacred Hindu temple dedicated to Lord Shiva |

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional installation required

### How to Use

1. **Clone or download** the project files
2. **Open** `Tourism.html` in your web browser
3. **Click "Get Started"** to explore favourite places
4. **Select any destination** to view detailed information with images
5. **Use the "Back" button** to return to previous sections
